## 基本操作
### 1.将文件上传到GitHub
~~~
git init
git add .
git commit -m ""
git remote add origin 仓库连接
git push -u origin main
~~~

### 2.查看提交日志
~~~
git log --pretty=oneline --all --graph --abbrev-commit
~~~

### 3.版本回退
~~~
git reset --hard commitID
~~~

这里的commitID可以通过上面的git log操作查看

那么我们如何查看已经删除的记录呢？
~~~
git reflog
~~~
在这里就可以看到commitID
